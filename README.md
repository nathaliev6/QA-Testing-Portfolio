# Agoda QA Testing Portfolio

This repository contains a personal **Manual QA practice project** based on exploratory and structured testing of the Agoda website.

The purpose of this project is to demonstrate practical QA skills including:

- Test scenario creation
- Test case design
- Bug reporting
- Product improvement suggestions
- Functional testing mindset
- Documentation and reporting

---

## Project Files

### `test_scenarios.csv`
Contains high-level testing scenarios covering key Agoda website modules such as:

- Login
- Search
- Booking
- Validation
- UI checks

### `test_cases.csv`
Contains detailed step-by-step manual test cases with:

- Preconditions
- Test steps
- Test data
- Expected results
- Priority

### `practice_bug_reports.csv`
Contains sample bug reports based on realistic QA testing situations such as:

- Validation issues
- Booking inconsistencies
- Search/filter issues
- UI observations

### `product_improvements.csv`
Contains suggested product / UX improvements identified during exploratory testing.

Examples:

- Better chatbot visibility
- Easier support access
- Improved user guidance
